# Getting Started

## Installation

## Prerequisites

- Python 3.6 or later
- pip 19.0 or later

## Installation Steps

- Clone the repository: git clone
  <https://github.com/jmuelbert/jm-dev-tools.git>
- Change into the directory: cd jm-dev-tools/
- Create a virtual environment: `bash python3 -m venv .venv`
- Activate the virtual python environment: `bash source .venv/bin/activate`
- Install the dependencies: `bash pip install hatch uv`
- Build the application: `bash hatch build`
- Run the program: `bash hatch run jm_bde`
